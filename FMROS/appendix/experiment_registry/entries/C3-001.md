# C3-001 — X1 composite confirmation on odd years

Program: P2+P5+P6 · State: closed · **Verdict: WEAKENED**

## Hypothesis
The frozen X1 composite (G2 barriers TP3/SL2×ATR/64 bars; HMM vol-gate
P>0.6 causal-filtered; limit entry δ=0.10×ATR, 4-bar cancel, 1.25bp fill
cost; pre-gap flatten; 10× lev cap; cycle-2 threshold protocol) has
positive net expectancy on test years never used in its construction's
evaluation (odd years 2015–2025).

## Mechanism
Conjunction of three separately validated levers, each attacking one
term of net expectancy (signal concentration / cost denominator / cost
paid). Ancestors: H-A1, H-B2(+B2v2), H-B1, X1 (all in registry).

## Pre-registered decision rule (frozen in script docstring before run)
CONFIRM if net expectancy > 0 AND day-block bootstrap p(≤0) < 0.05;
WEAKEN if expectancy > 0 but p ≥ 0.05; REFUTE if expectancy ≤ 0.
Secondary (reported, non-gating): year-block bootstrap, per-year table,
pilot comparison.

## Design notes
Nested walk-forward: models retrained per odd test year on strictly
prior data (purge 96 bars); HMM gates refit likewise; same code paths as
cycle-2 (imports, not copies). Known residual caveat declared up front:
odd years influenced cycle-2 only via later folds' training sets and via
the composite's design motivation; 2027+ remains the clean arbiter.

## Results (`research/cycle3/results/c3_x1_confirmation.json`)
* 334 trades, 0.21/day; expectancy **+0.079R**; win 45.5%; PF 1.15;
  Sharpe 0.50; maxDD −23.8%; final equity 1.27× over the 6 test years.
* Day-block bootstrap: CI [−0.040, +0.204], **p(≤0) = 0.108** → misses
  the 0.05 gate.
* Year-block: CI [−0.091, +0.299], p = 0.187.
* By year: 2015 −0.21 (n=6) · 2017 +0.45 (n=45) · 2019 +0.21 (n=8) ·
  2021 +0.17 (n=114) · 2023 −0.11 (n=118) · 2025 −0.04 (n=43).

## Verdict application
Expectancy > 0, p ≥ 0.05 → **WEAKENED**, exactly per rule. Not confirmed;
not refuted. Point estimate is ~60% of the even-year pilot (+0.128R) —
consistent with a real effect plus winner's-curse shrinkage, and also
consistent with zero at this sample size.

## Diagnosis
Profit concentration again tracks vol regimes (2017, 2021 positive;
2023 — a high-trade but choppy year — negative). Pooled even+odd
descriptive expectancy ≈ +0.106R over 748 trades, but pooling was not
pre-registered and even-year data is exploratory-tainted; it is reported
here as description only.

## Lessons / spawned entries
* Structural patience requirement: vol-gated strategies need many regime
  episodes for year-level significance → C3-006 (standing 2027+ re-run,
  rule inherited verbatim).
* The 1.25bp passive-cost assumption is now the single most
  verdict-sensitive assumption → C3-002 (shadow spread/fill measurement).
