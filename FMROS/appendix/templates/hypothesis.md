# Hypothesis Template

```
ID:            (program prefix + sequence, e.g. C3-007)
Program:       (P1–P10)
State:         PROPOSED
Proposed by / date:

HYPOTHESIS
  One falsifiable sentence.

MECHANISM
  Whose behaviour creates the effect; why it isn't already arbitraged;
  what would make it disappear. (No mechanism → not accepted as an entry;
  route via the declared-universe mining track instead.)

ECONOMIC TERM ATTACKED
  gross signal / cost paid / cost denominator / risk shape / other.

EXPECTED EFFECT SIZE
  In R per trade, bp, or AUC — with the reasoning. (Auditor compares
  later against realized.)

FALSIFICATION CRITERION
  The observation that kills it, stated numerically.

MULTIPLICITY UNIVERSE
  What family of variants this belongs to; how many will be/could be
  tried; how deflation will work.

DATA REQUIREMENTS
  Slices to be consumed (check registry burn list first); any new data.

PRIOR ART
  Registry ancestors (esp. REFUTED ones — justify why this isn't a
  re-litigation); literature (as hypothesis source only).
```
